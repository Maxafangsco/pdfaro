# PDFaro SEO Map

## Route Structure

| Route pattern | Type | Pages |
|---|---|---|
| `/[locale]` | SSG homepage | 13 locales |
| `/[locale]/tools` | SSG tools directory | 13 locales |
| `/[locale]/tools/[tool]` | SSG tool pages | ~99 tools × 13 locales |
| `/[locale]/tools/category/[category]` | SSG category pages | per locale |
| `/[locale]/about`, `/faq`, `/privacy`, `/contact` | SSG static pages | 13 locales |
| `/[locale]/workflow` | SSG workflow editor | 13 locales |

**Router**: Next.js 15 App Router  
**Rendering**: `output: 'export'` — all pages are statically pre-rendered at build time  
**i18n**: next-intl, 13 locales (en, ja, ko, es, fr, de, zh, zh-TW, pt, ar, it, id, vi)

---

## Metadata Implementation

All metadata is generated via `generateMetadata()` in each page file. The shared utility is at `src/lib/seo/metadata.ts`.

Every page includes:
- `title` (template: `%s | PDFaro`)
- `description`
- `alternates.canonical` — absolute URL with locale prefix
- `alternates.languages` — hreflang for all 13 locales
- `robots: { index: true, follow: true }`
- `openGraph` — title, description, url, type, images
- `twitter` — card, title, description
- JSON-LD structured data — SoftwareApplication, FAQ, HowTo, Breadcrumb schemas

## Crawling & Indexing

**robots.txt** — generated by `src/app/robots.ts`:
- `*` allow all, disallow `/api/`, `/_next/`, `/static/`
- Explicit allow for `GPTBot`, `ClaudeBot`, `PerplexityBot`
- Sitemap pointer: `https://www.pdfaro.com/sitemap.xml`

**sitemap.xml** — generated by `src/app/sitemap.ts`:
- All 13 locales × (6 static pages + ~99 tool pages)
- Priorities: homepage 1.0, /tools 0.9, tool pages 0.8, static 0.6
- changeFrequency: homepage daily, tools weekly, static monthly
